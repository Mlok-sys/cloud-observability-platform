🚀 Cloud Observability Platform

This project demonstrates a full monitoring and observability stack built on AWS EC2 using Docker, Terraform, and GitHub Actions.

🔍 Observability Stack

Grafana

Visualizes dashboards from Prometheus and Loki. Used for real-time monitoring and alerting.

Prometheus

Scrapes and stores metrics from Node Exporter and the app. Provides query-based insights.

Loki

Centralizes and indexes container logs for easy debugging alongside metrics.
