Cloud Observability Platform

This application is fully monitored through Grafana, Prometheus, and Loki. The project demonstrates a complete cloud-native observability stack built on AWS EC2, deployed using Terraform and automated via GitHub Actions.

Observability Stack

Grafana

Visualizes metrics and logs in real time from Prometheus and Loki.

Prometheus

Scrapes and stores metrics from Node Exporter and the app for monitoring and alerting.

Loki

Aggregates and indexes container logs for observability and debugging.

CI/CD Pipeline

This project uses an automated deployment pipeline built with GitHub Actions. On every push to the main branch:

  1. 1. Starts the AWS EC2 instance automatically.
  2. 2. Copies updated monitoring and app directories to the instance.
  3. 3. Rebuilds and redeploys containers with Docker Compose.
  4. 4. Verifies all observability services are running successfully.
