🚀 Cloud Observability Platform

This project demonstrates a complete observability stack on AWS EC2 using Docker, Terraform, and GitHub Actions.

🔍 Observability Stack

Grafana

Visualizes real-time dashboards from Prometheus metrics and Loki logs, providing deep observability insights.

Prometheus

Scrapes metrics from Node Exporter and the app, acting as the central time-series database for monitoring.

Loki

Collects and indexes container logs, allowing developers to correlate metrics and logs seamlessly.

🏗️ Project Architecture

The platform is built with Terraform-managed infrastructure, Docker-based monitoring stack, and CI/CD deployment via GitHub Actions.
