Cloud Observability Platform

This project demonstrates a full monitoring and observability stack built on AWS EC2 using Docker, Terraform, and GitHub Actions.

Observability Stack

Grafana

Visualizes dashboards from Prometheus and Loki for real-time system monitoring and alerting.

Prometheus

Scrapes and stores metrics from Node Exporter and the app, providing query-based insights.

Loki

Collects and indexes container logs for fast, centralized debugging.
